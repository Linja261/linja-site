So nutzt du den Tracker

  • • Während oder direkt nach dem Gespräch ausfüllen — nicht Tage später
  • • Daten bleiben lokal im Browser (sessionStorage-Auth + localStorage-Daten) — kein Server
  • • Per Export JSON sicherst du alles für ein anderes Gerät oder ins Vault
  • • Pro Interview 1 Datensatz pro Datum + Name — editierbar über „Bearbeiten" in der Liste
