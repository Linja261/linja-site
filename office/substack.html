Wie das funktioniert

  1. Scheduled Task publiziert Artikel automatisch auf linja.me und fügt einen Eintrag in diese Queue ein.
  2. Du bekommst Notification → öffnest diese Seite (Mobile o.k.).
  3. Copy Markdown → Open Substack → New Post → Paste → Publish.
  4. Posted-Button blendet den Eintrag aus (lokal gespeichert).
